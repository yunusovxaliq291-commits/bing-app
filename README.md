# 🎬 Bing — Video & Mesaj Platformu

## Quraşdırma

### Tələblər
- Node.js (nodejs.org saytından yükləyin)

### İşə salma (Windows/Mac/Linux)
1. Bu qovluğu açın
2. `start.bat` (Windows) və ya `start.sh` (Mac/Linux) faylını işə salın
3. Brauzerinizdə http://localhost:3000 açın

## Pulsuz onlayn hosting (hamı girə bilsin)

### Render.com (tövsiyə edilir, pulsuz)
1. https://render.com qeydiyyatdan keçin
2. "New Web Service" seçin
3. Bu qovluğu GitHub-a yükləyin, sonra Render-ə bağlayın
4. Build command: `npm install`
5. Start command: `node server.js`

### Railway.app
1. https://railway.app
2. "Deploy from GitHub" seçin

## APK (Android)
1. https://webintoapp.com açın
2. Hostinq URL-ni daxil edin
3. APK yükləyin

## EXE (Windows)
1. https://nativefier.app və ya
2. Electron ilə paketləmək üçün README-yə baxın
